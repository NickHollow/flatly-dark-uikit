# flatly-dark-uikit
a flat, dark, simple uikit theme

this repository is based on the uikit/uikit repository.


## Getting started

```
git clone https://github.com/calben/flatly-dark-uikit
cd flatly-dark-uikit
npm install
gulp -t flatly-dark
```
